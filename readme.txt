This is a simple Django API for a Blog. It has CRUD functionality with JWT authentication and dj-rest-auth registration/login.
The Requirements for this project are given in the Requirements.txt file.
Use of Postman app is recommended.

The commands for this project are:
To watch all posts: http://127.0.0.1:8000/blogpost/allpost/
To create a new post: http://127.0.0.1:8000/blogpost/allpost/
To update a post: http://127.0.0.1:8000/blogpost/post/(int)
To delete a post: http://127.0.0.1:8000/blogpost/post/(int)
To register a user: http://127.0.0.1:8000/accounts/signup/
To Login: http://127.0.0.1:8000/accounts/login/ or http://127.0.0.1:8000/auth/login/
To Logout: http://127.0.0.1:8000/accounts/logout/
To generate JWT Token: http://127.0.0.1:8000/blogpost/api/token/
